谜题写在txt文件中，可以作为参数来运行：

    $ cd $GOPATH/src/github.com/LauTrond/sudoku
    $ go run . puzzles/simple-01.txt

使用 -h 参数显示命令选项：

    $ cd $GOPATH/src/github.com/LauTrond/sudoku
    $ go run . -h

puzzles/hard-02.txt 是Google找到的号称"最难的数独"，耗时约15毫秒：

    $ cd $GOPATH/src/github.com/LauTrond/sudoku
