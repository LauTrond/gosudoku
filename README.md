# sudoku 数独解题

毫秒级的数独解题，还能显示每一步的推断。

试试：

    go get github.com/LauTrond/sudoku
    echo "
       7
    1
       43 2
            6
       5 9
          418
        81
      2    5
     4    3
    " | go run github.com/LauTrond/sudoku
